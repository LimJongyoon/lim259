  en: |
    Client-based freelance web projects including PW Company and Our Beloved Church, focusing on community services and internal operational management. The work covers responsive UX design and end-to-end full-stack web development across multiple real-world deployment environments.

    Representative systems include an ERP-oriented internal management platform designed to centrally manage staff information, participation records, and operational performance within a controlled organizational environment. The architecture places strong emphasis on security and operational stability, incorporating IP-based access control, user activity tracking, and comprehensive logging mechanisms.

    In addition, a hospital-oriented clinical data management web application was developed for internal use within the Korea University Hospital Plastic Surgery Department’s closed medical network. The system focuses on structured clinical records, patient-associated data, and image-based documentation, prioritizing data integrity, access control, and long-term maintainability over real-time public interaction.

    PW 컴퍼니, 사랑하는 우리교회 등을 대상으로 커뮤니티 서비스 및 내부 운영 관리를 중심으로 수행한 개인 외주 웹 프로젝트들이다. 반응형 UX 설계와 풀스택 웹 개발 전반을 포함하며, 실제 운영 환경에 배포되는 시스템 구축을 기준으로 개발되었다.

    대표 사례로는 직원 정보, 참여 현황, 운영 성과를 조직 내부 환경에서 통합적으로 관리하기 위해 설계된 ERP 성격의 사내 관리 시스템이 있으며, 보안성과 운영 안정성을 핵심 설계 기준으로 삼아 IP 기반 접근 제어, 사용자 활동 추적, 포괄적인 로그 관리 구조를 포함하도록 아키텍처가 구성되었다.

    또한 고려대학병원 성형외과 내부망에서 사용되는 임상 데이터 관리용 웹 애플리케이션 개발을 수행하였다. 해당 시스템은 환자 연관 데이터, 구조화된 임상 정보, 이미지 기반 기록을 안전하게 관리하는 것을 목표로 하며, 실시간성보다는 데이터 무결성, 접근 통제, 장기적인 운용 안정성에 중점을 두고 설계되었다.
