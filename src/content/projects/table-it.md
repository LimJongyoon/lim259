body:
  en: |
    Table-It is an interactive AR table system that transforms a physical table into a large-scale interactive interface using vision-based motion and color recognition. The project received the Korean Government Minister’s Award at EFESTA 2017.

    Hand motion recognition enables users to control a cursor and perform click actions by opening and closing their hands, effectively replicating standard computer interactions. In addition, color recognition allows the system to support applications such as table tennis and billiards, which are difficult to implement on conventional tablets.

    This project demonstrates how computer vision and projection-based AR can extend everyday surfaces into interactive computing platforms, emphasizing natural interaction and shared physical experiences.

  kr: |
    Table-It은 비전 기반 동작 인식과 색상 인식을 활용하여 물리적인 테이블을 대형 인터랙티브 인터페이스로 확장한 AR 테이블 시스템이다. 본 프로젝트는 EFESTA 2017에서 대한민국 정부 장관상을 수상하였다.

    손의 개폐 동작을 인식하여 마우스 커서 이동 및 클릭을 구현함으로써 일반적인 컴퓨터 조작을 자연스럽게 재현하였다. 또한 색상 인식을 통해 탁구, 당구와 같은 기존 태블릿 환경에서는 구현이 어려운 콘텐츠를 테이블 위에서 직접 체험할 수 있도록 설계되었다.

    본 프로젝트는 컴퓨터 비전과 프로젝션 기반 AR 기술을 활용하여 일상적인 사물을 인터랙티브 컴퓨팅 플랫폼으로 확장하는 가능성을 제시한다.
