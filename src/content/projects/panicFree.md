  en: |
    panicFree is a hybrid mobile application designed for structured daily logging and long-term tracking of personal mental and behavioral states. By translating subjective experiences into consistent records over time, the system enables users and observers to identify patterns, changes, and trends that are difficult to perceive through memory alone.

    Beyond the user-facing interface, the project includes the design and implementation of server-side logic written in JavaScript-based environments, handling data storage, validation, and time-based aggregation of user records. This backend layer supports continuous data accumulation and reliable longitudinal analysis across extended usage periods.

    The application emphasizes continuity, simplicity, and repeated engagement, positioning daily self-reporting not as a diagnostic act but as a reflective practice that connects everyday experience with structured observation and analysis.

    사용자 인터페이스 설계와 함께, 본 프로젝트는 JavaScript 기반 환경에서 작성된 서버 사이드 로직을 포함하며, 데이터 저장, 검증, 시간 축을 고려한 기록 누적 및 관리 구조를 직접 설계·구현하였다. 이를 통해 장기간 사용 시에도 데이터의 연속성과 분석 가능성을 안정적으로 유지하도록 구성되었다.

    본 시스템은 진단을 목적으로 하기보다, 반복적 기록과 지속적 사용을 통해 개인의 상태를 구조적으로 바라볼 수 있는 관찰 도구로서의 역할에 초점을 두며, 일상적 경험과 체계적 분석 사이를 자연스럽게 연결하도록 설계되었다.
